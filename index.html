ABTdomain Open Source

Domain Intelligence · Domain Monitoring · Brand Protection

ABTdomain is a domain intelligence platform that tracks domain lifecycle changes across the internet. We monitor over one million domain events daily — new registrations, expirations, ownership transfers, and DNS changes — to provide structured, verifiable intelligence for cybersecurity professionals, brand protection teams, and developers working with domain data.

This page showcases our open-source tools and MCP integrations. Below you will find our word segmentation libraries for analyzing domain names programmatically, and our MCP server that brings domain intelligence directly into AI-powered development workflows.

Open Source Projects

These tools were built from real problems we encountered while processing and analyzing millions of domain names. We open-sourced them so that developers, security researchers, and NLP engineers can use them in their own projects.

Python

DKSplit

Word segmentation library that splits concatenated strings into meaningful words. Built with a BiLSTM-CRF model trained on millions of labeled samples covering domain names, brand names, and multi-language phrases. The model is exported to ONNX format with INT8 quantization for fast, lightweight inference. Useful for domain name parsing, keyword extraction from URLs, SEO analysis, and brand monitoring.

GitHub → PyPI →
Go

DKSplit-Go

Go implementation of DKSplit for high-throughput environments. Handles up to 7,000 queries per second in batch mode and runs 2.7x faster than the Python version in single-query mode. Designed for backend services, domain processing pipelines, and applications where low latency and high concurrency matter.

GitHub → pkg.go.dev →
MCP

DomainKits MCP

Model Context Protocol server that connects ABTdomain's domain intelligence data to AI assistants. Search newly registered domains by keyword, perform nameserver reverse lookups, and query domain statistics — all from within Claude, Cursor, or any MCP-compatible client. Built for developers who want to integrate domain data into AI-powered workflows without building custom integrations.
